ALMA Archive Query Tool

alma_nearby_search.py

Query the ALMA archive around a list of sky positions. Check what bands and arrays were used, whether common molecular lines are covered, and export everything to CSV.

Running the script

Usage

Basic invocation — input file, output file, done.

bash 
$ python alma_nearby_search.py input.csv output.csv

Options

bash · examples 
# Tighten the search radius to 3 arcmin
$ python alma_nearby_search.py in.csv out.csv --radius-arcmin 3

# Disable deduplication
$ python alma_nearby_search.py in.csv out.csv --dedup-level none

# Query a different species
$ python alma_nearby_search.py in.csv out.csv --observed-species HCN

# Custom species thresholds
$ python alma_nearby_search.py in.csv out.csv --observed-species HCN \
  --observed-distance-threshold-arcsec 20 \
  --observed-fov-threshold-arcsec 80

# Compact output (cleaner mode)
$ python alma_nearby_search.py in.csv out.csv --cleaner

# Verbose logging
$ python alma_nearby_search.py in.csv out.csv --verbose

How it works

1

TAP query

Queries ivoa.obscore through the ALMA TAP service for each input target within a cone search.

2

Separation

Recomputes target-to-observation angular separation precisely with astropy.

3

Spectral parsing

Parses ALMA spectral coverage from frequency_support when available.

4

Line inference

Infers likely covered transitions using the built-in line catalog.

5

Array classification

Tags each observation as 12m, 7m, and/or TP array.

6

Deduplication & output

Writes one row per surviving match. Sources with no nearby ALMA data still get a NaN row.

The Observed X in ALMA? Flag

Assigned at the source level — all rows with the same Name share the same final value. The column name follows --observed-species (e.g. Observed HCN in ALMA?).

1

Species present and at least one row within the distance threshold

0.5

Species present, no score-1 row, but FOV covers the source from farther away

0

Neither condition met — no coverage for the selected species

If both 1 and 0.5 conditions exist for the same source, the final value is 1.

For --observed-species CO, matching is limited to the CO family (12CO, 13CO, C18O, C17O). Unrelated species like H₂CO or HCO⁺ are not counted.

Cleaning up

Deduplication

Multiple spectral windows or execution blocks within one project can produce repeated rows. Use --dedup-level to control this.

LevelBehaviour
noneAll rows kept as-is
projectOne row per ALMA project code per source
project_target defaultOne row per project code and ALMA target name per source

Cleaner Mode

For a compact, publication-ready output:

bash 
$ python alma_nearby_search.py in.csv out.csv --cleaner

# Limit to 3 rows of the target species per source
$ python alma_nearby_search.py in.csv out.csv --cleaner --cleaner-max-observed-rows-per-name 3

Cleaner rules, in order:

  • Keep one NaN row when a source has no ALMA match.
  • Keep up to 5 closest rows containing the selected species per source (configurable).
  • If the species is absent, keep the single closest row for that source.
Built-in catalog

Identifiable Lines

Lines are inferred from spectral coverage — not from detections. The catalog covers the most common extragalactic and galactic tracers.

CO Family
  • ¹²CO (1–0) through (7–6)
  • ¹³CO (1–0) through (4–3)
  • C¹⁸O (1–0) through (4–3)
  • C¹⁷O (1–0) through (3–2)
Dense Gas
  • HCN (1–0), (3–2), (4–3)
  • HCO⁺ (1–0), (3–2), (4–3)
  • HNC (1–0), (3–2), (4–3)
  • N₂H⁺ (1–0), (3–2)
  • CS (2–1), (3–2), (5–4), (7–6)
PDR / Diffuse Gas
  • CN (1–0), (2–1)
  • CCH (1–0), (3–2)
Atomic Carbon
  • [CI] ³P₁–³P₀
  • [CI] ³P₂–³P₁
Shock Tracers
  • SiO (2–1), (5–4), (8–7)
  • SO (5₆–4₅), (6₅–5₄)
  • SO₂ (11₁,₁₁–10₀,₁₀)
Chemistry / Cold Gas
  • H₂CO (3₀₃–2₀₂)
  • H₂CO (3₂₂–2₂₁)
  • H₂CO (3₂₁–2₂₀)
  • H₂CO (2₁₂–1₁₁)
  • NH₂D (1₁₁–1₀₁)
Star-Forming / Complex
  • CH₃OH (5₀–4₀)
  • CH₃OH (2k–1k)
  • CH₃CN (12₀–11₀)
Recombination
  • H30α
  • H40α

Some entries (CN, CCH, CH₃OH 2k–1k) are group-center proxies for blended or multi-component spectral complexes.
