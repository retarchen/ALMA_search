ALMA Search documentation
=========================

``alma_search`` is a Python package and command-line tool for querying the ALMA
archive around a list of sky positions and exporting nearby observations to CSV.
It is designed for small target catalogs where you want to inspect ALMA
coverage, receiver bands, array configurations, and common line coverage such
as CO or HCN.
